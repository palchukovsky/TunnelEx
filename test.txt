===============================

     Тестирование TunnelEx     

===============================



1. Общие unit-тесты
===================
Для запуска общих тестов необходимо запустить на выполнение проект "Test",
либо исполняемый файл "TexTest.exe".




2. Тестирование передачи данных
===============================
Для тестирования туннеля необходимо:
   1. запустить TunnelEx и добавить необходимые правила
   2. запустить тест-сервер
   3. запустить тест-клинета

Примечание: при передачи команды тире можно опускать и набирать слова
команды слитно.

1.2. TCP
--------
    тест-сервер: TexTest.exe tcp-server [ port ]
    тест-клиент: TexTest.exe tcp-client host:port
    порт тест-сервера по умолчанию: 12332

1.3. UDP
--------
    тест-сервер: TexTest.exe udp-server [ port ]
    тест-клиент: TexTest.exe udp-client host:port
    порт тест-сервера по умолчанию: 12333

1.4. Пайп
---------
    тест-сервер: TexTest.exe pipe-server [ pipe ]
    тест-клиент: TexTest.exe pipe-client pipe
    имя пайпа тест-сервера по умолчанию: TunnelEx/Test/89326B80-AE04-11E0-9F1C-0800200C9A66




2. Полезные ключи
=================

--gtest_break_on_failure - остановка на первой ошибке
